case class PaymentRequest(prefix: String, amount: Option[MilliSatoshi], timestamp: Long, nodeId: PublicKey, tags: List[Tag], signature: BinaryData) extends Product with Serializable

Lightning Payment Request see https://github.com/lightningnetwork/lightning-rfc/pull/183

prefix

currency prefix; lnbc for bitcoin, lntb for bitcoin testnet

amount

amount to pay (empty string means no amount is specified)

timestamp

request timestamp (UNIX format)

nodeId

id of the node emitting the payment request

tags

payment tags; must include a single PaymentHash tag

signature

request signature that will be checked against node id
